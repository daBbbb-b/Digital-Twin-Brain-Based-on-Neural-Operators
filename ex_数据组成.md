## 数据说明

以下是该数据各个组成部分的详细解释：

### 1. 时间序列数据 (Time Series Data)

这部分包含了模拟的核心输出结果。

* **`'time_points'`** :
* **内容** : 一个包含 4000 个浮点数的数组。
* **范围** : 从 `0.00` 到 `199.95`。
* **含义** : 模拟的时间轴。结合 metadata 可知，总时长为 200ms (或秒，取决于单位定义)，采样间隔为 0.05。
* **`'bold_signal'`** :
* **形状** : `(4000, 246)`。
* **含义** : 模拟生成的  **BOLD 信号** （血氧水平依赖信号，常用于 fMRI）。
  *  **行 (4000)** : 对应 `time_points` 中的每一个时间步。
  *  **列 (246)** : 对应大脑的  **246 个感兴趣区域 (ROIs)** （例如使用 Brainnetome Atlas 图谱）。
* **数值** : 具体的信号强度值。注意数据末尾出现了 `nan`，这可能意味着模拟在某处发散、数据被截断或者这只是一个用于展示格式的片段。

### 2. 刺激配置 (`'stimulus_config'`)

这部分描述了在模拟过程中如何对大脑施加外部输入（任务刺激）。

* **`'type'`: `'task_based_ode'`** : 表明这是一个基于常微分方程（ODE）的任务态模拟。
* **`'n_channels'`: 246** : 对应大脑区域数量。
* **`'tasks'`** : 一个列表，包含 20 个具体的刺激事件（Index 0 到 19）。每个事件包含：
* **`'index'`** : 任务编号。
* **`'range'`** : `(开始时间, 结束时间)`。例如任务 0 发生在 2.30 到 8.45 时间段内。
* **`'channels'`** : 受到刺激的大脑区域索引列表。例如任务 0 刺激了第 39, 112, 167 号区域。
* **`'amplitudes'`** : 对应每个通道的刺激强度。正值通常代表兴奋性输入，负值代表抑制性输入。

### 3. 模型参数 (`'model_params'`)

这部分定义了用于生成数据的生物物理模型的参数。

* **时间常数** :
* `'tau_E'`: 0.01 (兴奋性种群的时间常数)。
* `'tau_I'`: 0.02 (抑制性种群的时间常数)。
* **耦合权重 (局部)** :
* `'w_EE'`, `'w_EI'`, `'w_IE'`, `'w_II'`: 定义了区域内部兴奋(E)和抑制(I)种群之间的相互作用强度。
* **全局耦合** :
* `'G'`: 1.0。全局耦合系数，用于缩放区域间的连接强度。
* **`'C'`** : 形状为 `(246, 246)` 的数组。这是  **结构连接矩阵 (Structural Connectivity Matrix)** ，通常源自 DTI (弥散张量成像) 数据，定义了 246 个脑区之间物理连接的拓扑结构（0代表无连接，1或其他值代表有连接）。
* **传递函数参数** :
* `'a_E'`, `'b_E'`, `'d_E'` 等：用于定义神经元群体的激活函数（通常是 Sigmoid 函数）的形状参数。

### 4. 初始状态 (`'initial_state'`)

* **内容** : 一个一维数组。
* **含义** : 微分方程在 `t=0` 时刻的初始值。
* **推测** : 由于是 E-I 模型且有 246 个区域，如果每个区域有两个状态变量（E 和 I 的发放率），数组长度通常是 246×2=492**246**×**2**=**492**。或者是经过某种扁平化处理的状态向量。

### 5. 元数据 (`'metadata'`)

这部分记录了模拟的控制参数，用于复现实验。

* **`'model_type'`: `'EI'`** : 确认模型类型为兴奋-抑制网络。
* **`'dt'`: 0.05** : 数值积分的时间步长。
* **`'duration'`: 200.0** : 模拟总时长。
* **`'sampling_interval'`: 0.05** : 数据保存的频率（这里与步长一致，说明没有降采样）。
* **`'noise_level'`: 0.02** : 模拟中加入的随机噪声强度。
* **`'noise_seed'`: 1000** : 随机数种子，确保模拟结果可重复。
